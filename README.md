# quantum_visualizer
MIPT Python project :)


Главная идея проекта

Создать **интерактивный пошаговый визуализатор квантовых алгоритмов** (Например, для алгоритма Шора для факторизации целых чисел, пользователь вводит число (например, `15`) и **сам управляет выполнением алгоритма** — нажатие клавиши выполняет следующий квантовый гейт или измерение). Можно будет пробовать показывать: **живую 3D-визуализацию состояния кубитов** (сферы Блоха), **графы квантовых цепей** и **интерференционную картину** Квантового Преобразования Фурье (QFT).

Цель — сделать сложный квантовый алгоритм **наглядным, понятным и красивым** для студентов, преподавателей и всех интересующихся квантовыми вычислениями.

---

Ключевые фичи (пока набросок идей, возможно что-то прибавится, а что-то, наоборот, уберём, если не будем успевать, пока с запасом):

1.  **Пошаговое управление** — пользователь сам решает, когда выполнить следующий шаг.
2.  **3D Сферы Блоха** — визуализация состояния каждого кубита в реальном времени с плавной анимацией поворота.
3.  **Визуализация запутанности** — граф взаимодействия кубитов, где рёбра показывают CNOT-гейты и меру запутанности.
4.  **Интерактивная QFT-диаграмма** — отображение вращений фаз как "стрелок на окружности".
5.  **Режим "Квантовый шум"** — ползунок, добавляющий визуальный шум к сферам Блоха, демонстрирующий проблемы NISQ-компьютеров.
6.  **Сравнение с классикой** — таймер, показывающий, сколько времени заняла бы факторизация классическим алгоритмом.
7.  **Встроенная теория** — отдельная вкладка/панель с объяснением каждого шага алгоритма простым языком.
8.  **Экспорт схемы** — возможность сохранить текущую квантовую схему в формате Qiskit (`.qpy`) или изображение схемы.
9.  **Авторизация** — возможность авторизоваться в аккаунт и сохранять прогресс.
10. **Создание схемы** — возможность самому создать квантовую схему и посмотреть как она работает.
11. **Internet sharing** — возможность коммуникации пользователей с разных аккаунтов с разных компьютеров.

Библиотеки:
1.  **Quskit** — Симуляция квантовых схем, расчёт векторов состояний
2.  **Ursina Engine** — 3D-рендеринг сфер Блоха и создание интерфейса (может ещё подумаем и сменим на pygame, но хотелось бы всё таки 3D визуализацию, так что надеемся, что успеем освоить эту библиотеку)
3.  **NetworkX** — Построение и визуализация графов квантовых цепей
